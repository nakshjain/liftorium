| Document | Purpose |
| --- | --- |
| [Prompt Log](./prompt-log.md) | Chronological record of important prompts |
| [Reusable Prompts](./reusable-prompts.md) | Stable prompts for implementation, review, testing, and documentation |

## Capture Guidelines

Save prompts that:

- Define feature scope.
- Establish architecture direction.
- Produce reusable implementation patterns.
- Help debug a meaningful issue.
- Guide code review or test planning.

Avoid saving low-value prompts such as minor typos, one-off command requests, or short clarifications.
